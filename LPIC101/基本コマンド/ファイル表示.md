 # head tail
   ファイルの先頭から表示する行数を-nオプションで指定（デフォルトで10行）
   -fオプションはログファイルのモニタに有効
   
# tr [オプション] [文字群1 [文字群2]]
  文字群1を[文字群2]の形式に変換して出力　※Ctrl + D　で終了
  
# sed [オプション] [編集 コマンド] [ファイル 名]
  単語の表示・変換・削除を行う
  
# cut
  ファイル内の行中の特定部分のみ取り出す
  
# wc [オプション] [ファイル 名]
  ファイル内のバイト数、単語数、行数を表示する
  
# split [オプション] [入力元 ファイル名 [プレフィックス]]
  ファイルを決まった大きさに分割
